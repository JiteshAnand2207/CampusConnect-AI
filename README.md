# CampusConnect AI

CampusConnect AI is a MERN-based college event management and campus issue reporting platform. It includes authentication, role-based dashboards, event registration, QR ticket verification, problem reporting, notifications, file uploads, and an AI assistant.

## Live Demo

https://campusconnect-ai-backend-node.onrender.com
![CI](https://github.com/JiteshAnand2207/campusconnect-ai/actions/workflows/ci.yml/badge.svg)
## Deployment

The project is deployed as a single full-stack Render service.

- React frontend is built using Vite.
- Express serves the production frontend from `client/dist`.
- Backend APIs are available under `/api`.
- MongoDB Atlas is used as the cloud database.
- Redis is optional and disabled in the current deployment.
- AI assistant currently runs in mock mode for safe deployment without paid API keys.
